IA · CLAVE DE GEMINI

El escáner de máquinas y el entrenador IA usan la API gratuita de Google Gemini. Para que estas funciones sean gratis para ti y para tus amigos, cada persona necesita su propia clave gratuita (no hace falta tarjeta de crédito).

  1. Abre aistudio.google.com/app/apikey
  2. Inicia sesión con tu cuenta de Google
  3. Pulsa "Create API key" y cópiala
  4. Pégala aquí abajo

🔒 Tu clave se guarda solo en este dispositivo (localStorage) y nunca se envía a ningún servidor de FitForce. Las peticiones van directas desde tu navegador a Google.
